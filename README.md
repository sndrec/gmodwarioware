# gmodwarioware

A gamemode quite heavily inspired by the original TF2Ware by Mecha the Slag.

This gamemode will change a few server values for suitable gameplay which may need to be changed back afterward:
sv_gravity
sv_accelerate
sv_airaccelerate
sv_friction
sv_stopspeed
mp_falldamage
sbox_noclip
